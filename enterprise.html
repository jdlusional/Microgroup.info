Enterprise Client Login

This demonstrates the sign-in experience a client sees, not a real login. Access to each organization's actual pages is enforced at the network edge by Cloudflare Access, scoped to that organization's own email domain — there is no password or session handled by this site's own code.

Each page here is built for one real organization, run against that organization's own funding profile, board, and public record, not an invented sample. The funders, foundations, trustees, grants, and dollar figures are real and come from public IRS filings.
